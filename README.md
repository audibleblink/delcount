# The Real Delegate Count

A simple site that shows the pledged delegate count, according to AP. No super-delegates here.

![](http://i.imgur.com/Bz36Fo6.png)


# Developing

Edit the files in `src` and run `gulp` to process them into `www`.

Repo owner: Push to live site with `git subtree push --prefix www origin gh-pages`


# Contributing
Please refer to each project's style guidelines and guidelines for submitting patches and additions. In general, we follow the "fork-and-pull" Git workflow.

 1. Fork the repo on GitHub
 2. Commit changes to a branch in your fork
 3. Pull request "upstream" with your changes
 4. Merge changes in to "upstream" repo
